Instructions on how to host website using Github on Github pages

  1. To host on github pages you need to sign in on Github and create a new repository.
  2. Set repository to public.
  3. Upload your Project with index.html file on main page.
  4. Go to Settings and scroll down to Github pages section.
  5. Click on sources button and set to main and then click save.
  6. Wollah! your static website is hosted.
